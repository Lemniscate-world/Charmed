# Alarmify Icon Design

## Concept
A modern, minimalist alarm clock icon inspired by Charm CLI aesthetics with a music twist.

## Icon Specifications

### Primary Icon (512x512)
- **Shape**: Circular base with rounded clock face
- **Colors**: 
  - Background: Gradient from #1DB954 (Spotify Green) to #1ED760
  - Clock hands: #FFFFFF with subtle shadow
  - Music note accent: #FFD700 (Gold)
  - Border: 2px solid rgba(255, 255, 255, 0.2)
- **Elements**:
  - Simplified clock face with minimal hour markers
  - Music note integrated with minute hand
  - Subtle glow effect around the perimeter

### Tray Icon (16x16, 32x32, 48x48)
- Simplified version with just clock outline and music note
- High contrast for visibility in system tray
- Monochrome variant for light/dark system themes

### Design Elements
- Border radius: 20% of icon size for modern rounded look
- Shadow: 0 4px 8px rgba(0, 0, 0, 0.3)
- Glassmorphism: Subtle inner highlight at top (rgba(255, 255, 255, 0.1))

## SVG Implementation
The icon will be generated programmatically as SVG for scalability and rendered to PNG for application use.
